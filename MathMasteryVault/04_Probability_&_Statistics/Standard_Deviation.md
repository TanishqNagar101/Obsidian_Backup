# 📄 Standard Deviation

## 🧠 Intuitive Summary
Standard deviation tells us how "spread out" the values are in a dataset.
Think of it like how wild or calm a party is — are people clustered in one room (low SD) or everywhere (high SD)?

## 🧮 Formal Definition / Formula
σ = sqrt( Σ(xᵢ - μ)² / N )

Where:
- xᵢ = each value
- μ = mean of the values
- N = number of values
