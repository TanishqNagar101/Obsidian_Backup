- Let's Suppose we have: -
		$\large{Orignal\ Equation \rightarrow Y = \beta_{0} + \beta_{1}X + \varepsilon}$                        1{Population Regression}
		$\large{Least-Square\ Equation \rightarrow \widehat{Y} = \hat{\beta}_{0}+\hat{\beta}_{1}X+\varepsilon}$         2{Least-Square Regression}
- The equ1 is is generally not known but from the sample of population we can always calculate equ2 from the sample.
- But if we could average a huge number of estimates of $\widehat{Y}$ obtained from a huge number of sets of observation, then this average is exactly equal to $Y$.
- If we use use the analogy of $\large{\mu}$ then to estimate the $\large{\hat{\mu}}$ we can use the following formula: -
		$\large{\hat{\mu} = \bar{y}\ \ then,}$ 
		$\large{\bar{y} = \frac{1}{n} \sum_{i}^{n} y_{i}}$ 
- Now we know that avg of $\large{\hat{\mu} = \mu}$ for a large number of set of observations.
- And, $\large{\hat{\mu}}$ of a single sample data is either underestimate or overestimate of $\large{\mu}$ . Now to calculate the amount by which it varies can be calculated using this formula: -
		$\large{Var(\hat{\mu}) = SE(\hat{\mu})^{2} = \frac{\sigma^{2}}{n}}$      $\large{\{SE \rightarrow Standard Error \}}$
