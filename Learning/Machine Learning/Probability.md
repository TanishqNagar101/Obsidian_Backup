- It is the study of claims in simple words i.e. we are trying to find out what are the chances of Event "E" happening within the given environment. It is denoted by *p* and read as *p*(x)<- probability of event "x" happening.
- Other definition is it assign a numerical measure to the uncertainty of events in a well-defined sample space.  
- Range 0<=*p*<=1.(Higher the number higher the chance of event happening)
- Statisticians use this to verify hypothesis, it can be used to check against some established fact or a standalone statement. The method use for this is called *p-value* (probability value). The method to use this is as follows-
	- Then we establish the base hypothesis ($H_{1}$) and an opposing hypothesis $(H_{0})$ and then we calculate the $p-value$ test if $p(H_{0})>0.05$ then ($H_{1}$) is rejected otherwise $H_{1}$ is accepted.
- Now there are three ways to calculate the probability :-
	a) Approximation - $p(x) = {No. of events x is present}/{Total no. of events}$   
	b) Classical approach(Require Equally likely events) = no of ways event occur / total no of simple events 
	c) Subjective probability - Determining the probablity using the knowledge of enviroment.
